把本页链接发给被控电脑用户即可。服务器地址和令牌已内置，设备 ID 默认为电脑名。

推荐：按 Win+R，粘贴下方一行命令并回车。无需下载 vbs/bat，较少触发 Windows 安全中心。

1复制上方命令，在被控电脑按 Win+R 粘贴回车
2若出现 Windows 安全提示，点「更多信息」→「仍要运行」
3安装完成后程序自动启动，日志见 %TEMP%\ReSA-install.log

关于 Windows 安全中心

ReSA 未做代码签名，首次安装时 Windows 可能提示「已保护你的电脑」或拦截脚本，这是正常现象。

处理方法：

1SmartScreen 蓝窗：点「更多信息」→「仍要运行」
2下载的 ps1：右键文件 → 属性 → 勾选「解除锁定」→ 确定
3不要使用 vbs 安装（更容易被安全中心拦截）
