RemoteScreen Agent 远程一键安装

把本页链接发给被控电脑用户即可。服务器地址和令牌已内置，设备 ID 默认为电脑名。

在被控 Windows 电脑上打开此页，点击下方按钮下载安装脚本，双击运行即可完成安装与开机自启。

下载一键安装脚本
1点击上方按钮，下载 RemoteScreenAgent-Install.bat
2双击运行（若 SmartScreen 提示，选「更多信息」→「仍要运行」，仅首次）
3回到控制端，在设备列表查看是否在线

发给对方的安装链接

高级：PowerShell 一行命令

无法下载 bat 时，在被控电脑 PowerShell 中粘贴运行：

安装后路径

程序：%LOCALAPPDATA%\RemoteScreenAgent\RemoteScreenAgent.exe

设备 ID：%LOCALAPPDATA%\RemoteScreenAgent\device.id

日志：%LOCALAPPDATA%\RemoteScreenAgent\agent.log

← 返回控制端
