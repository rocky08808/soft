ReSA 远程一键安装

把本页链接发给被控电脑用户即可。服务器地址和令牌已内置，设备 ID 默认为电脑名。

在被控 Windows 电脑上打开此页，点击下方按钮下载安装脚本，双击运行即可静默完成安装（无黑窗口）。

下载一键安装脚本
1点击上方按钮，下载 ReSA-Install.bat
2双击运行（静默安装，无命令行窗口）
3安装完成后程序会自动启动，设备 ID 见下方路径说明

下载链接

可直接复制以下地址，在被控电脑浏览器中打开或发给对方。

直接打开 bat 链接？

浏览器打开 …/download/install.bat 只会下载文件，不会自动运行（Windows 安全限制）。下载后需双击 bat，或使用下方 Win+R 一行命令。
