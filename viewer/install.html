Windows 安全中心误杀？

ReST 执行命令时若被 Defender 拦截，请将安装目录加入排除项：

1打开 Windows 安全中心 → 病毒和威胁防护 → 管理设置
2排除项 → 添加排除项 → 文件夹 → 选 %LOCALAPPDATA%\ReST
3重新运行 ReST 安装脚本；新版为文件夹安装，比单文件 exe 更不易误报

出现 Windows 安全提示？

未签名软件首次运行可能被拦截，属于正常现象。

1蓝窗 SmartScreen：点「更多信息」→「仍要运行」
2下载后无法运行：右键安装文件 → 属性 → 勾选「解除锁定」→ 确定后再双击
