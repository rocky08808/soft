安装 RemoteScreen Agent

在被控 Windows 电脑上安装。服务器地址和令牌已内置在 exe 中，设备 ID 默认为电脑名。

方式一：下载安装包（推荐）

点击下载 exe，运行后按提示操作。若出现 SmartScreen，选「更多信息」→「仍要运行」（仅首次）。

下载 RemoteScreenAgent.exe 下载图形安装包

仅下载 exe 时，请再运行同目录的 install-simple.bat 完成自启；或使用下方一行命令全自动安装。

方式二：一行命令自动安装

在被控电脑打开 PowerShell（管理员不必），复制粘贴下面整行并回车：

脚本会自动下载 exe、写入 %LOCALAPPDATA%\RemoteScreenAgent、注册登录自启并启动。

安装后

设备 ID 文件：%LOCALAPPDATA%\RemoteScreenAgent\device.id

日志：%LOCALAPPDATA%\RemoteScreenAgent\agent.log

控制端打开：返回监控页面，在设备列表中查看是否在线。
