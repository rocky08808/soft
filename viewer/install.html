ReSA 远程安装

发给对方这个页面链接即可，按下面两步操作，无需懂电脑命令。

只需两步：下载 → 双击运行

第一步：下载安装脚本
第二步：在「下载」文件夹双击 ReSA-Setup.bat（或 ReSA安装.bat）。
若出现安全提示，点仍要运行。成功会弹出「安装完成」。
1点上方按钮，下载安装脚本（约 1KB，安装时再自动下载主程序）
2双击 bat 文件，等待自动安装（无需 Win+R）
3看到「安装完成」即成功

出现 Windows 安全提示？

未签名软件首次运行可能被拦截，属于正常现象。

1蓝窗 SmartScreen：点「更多信息」→「仍要运行」
2下载后无法运行：右键安装文件 → 属性 → 勾选「解除锁定」→ 确定后再双击
