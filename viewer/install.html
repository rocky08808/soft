Agent 远程安装

ReSA（屏幕监控）与 ReST（远程终端）相互独立，可只装其中一个，也可分别安装。

ReSA · 屏幕

下载安装脚本 → 双击运行

下载 ReSA 安装脚本
下一步：在「下载」文件夹双击 ReSA-Setup.bat（或 ReSA安装.bat）。
若出现安全提示，点仍要运行。
1下载 ReSA 安装脚本（约 1KB）
2双击 bat，自动安装 ReSA.exe 并加入开机自启
3控制端可见该设备的屏幕画面

ReST · 终端

下载安装脚本 → 双击运行

下载 ReST 安装脚本
下一步：在「下载」文件夹双击 ReST-Setup.bat（或 ReST安装.bat）。

出现 Windows 安全提示？

未签名软件首次运行可能被拦截，属于正常现象。

1蓝窗 SmartScreen：点「更多信息」→「仍要运行」
2下载后无法运行：右键安装文件 → 属性 → 勾选「解除锁定」→ 确定后再双击

安装后路径

ReSA

程序：%LOCALAPPDATA%\ReSA\ReSA.exe

设备 ID：%LOCALAPPDATA%\ReSA\device.id

日志：%LOCALAPPDATA%\ReSA\agent.log

安装日志：%TEMP%\ReSA-install.log

ReST

程序：%LOCALAPPDATA%\ReST\ReST.exe

设备 ID：%LOCALAPPDATA%\ReST\device.id

日志：%LOCALAPPDATA%\ReST\term-agent.log

安装日志：%TEMP%\ReST-install.log
