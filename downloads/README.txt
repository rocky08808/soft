Online install files (upload to server downloads/):

  ReSA.exe            — screen agent (~50-70MB after build optimization)
  ReST.exe            — terminal agent (term-agent\build.bat, ~8-12MB)
  install.ps1         — ReSA silent install script
  install-rest.ps1    — ReST silent install script
  uninstall.ps1       — ReSA uninstall script
  uninstall-rest.ps1  — ReST uninstall script

Install launchers (generated by server, not stored here):
  ReSA-Setup.bat      — /download/ReSA-Setup.bat (~1KB)
  ReST-Setup.bat      — /download/ReST-Setup.bat (~1KB)

ReSA and ReST are independent:
  ReSA -> %LOCALAPPDATA%\ReSA\
  ReST -> %LOCALAPPDATA%\ReST\
  Each has its own device.id, autostart, and uninstall script.

Build:
  agent\build.bat       — builds ReSA.exe
  term-agent\build.bat  — builds ReST.exe
