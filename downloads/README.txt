Online install files (upload to server downloads/):

  ReSA.exe          — screen agent (~50-70MB after build optimization)
  ReST.exe          — terminal agent (term-agent\build.bat, ~8-12MB)
  install.ps1       — silent install script (server also generates ReSA-Install.ps1)
  uninstall.ps1     — uninstall script

Install launcher (generated by server, not stored here):
  ReSA-Setup.bat    — user downloads from /download/ReSA-Setup.bat (~1KB)

Build:
  agent\build.bat       — builds ReSA.exe
  term-agent\build.bat  — builds ReST.exe (uses same agent.config.json)

Public URLs:
  https://olxp.cc/install.html
  https://olxp.cc/download/ReSA-Setup.bat
  https://olxp.cc/download/ReSA.exe
